# Task Checklist

Simple Next.js app with Tailwind CSS. Features:
- Add tasks with input + button
- Toggle completion with checkbox (applies strike-through)
- Delete tasks
- Persist tasks in `localStorage`

Run locally:

```powershell
cd "c:\Users\yourb\OneDrive\Git\VideCoding"
npm install
npm run dev
```

Open http://localhost:3000
